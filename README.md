# Netflix-Clone (DEMO URL) : https://netflix-clone-2dd79.web.app/

•	Made a Netflix Clone with React.Js and Node and Hosted on FireBase.  
•	Used TMDB (Movie Database API) to get all the Movies Data.
•	User can see Movie Trailers and The API Data is live, so it changes every time page is Refreshed.
